Question:

 

  • Note:

 

            The directory files contains file_sum1.txt,file_sum2.txt ... file_sum10.txt

 

            These text files contain strings separated by new line 

 

            The first line consists of a string which is the actual text

 

            And the rest of the lines contains the hexadecimal value 

 

  •  Define a pytest fixture 'check_hex' which takes a file name as a parameter and does the following:

    

            Read the file provided in the filename and return the contents of the file.

 

  •  Define a function called encoder which can take a string value as a parameter and return the encoded hexadecimal value as a list

            If the length of the string is not an even number then raise a value error of 

            "not an even number of strings"

            or else perform the encoding in the following manner :

            for every two letters perform encoding using this function "str".encode('utf-8').hex()

            and append them to a list

            return the list

 

  •  Define a function called decoder which can take a list of hexadecimal values and perform decoding using the function bytes.fromhex("str").decode('utf-8')

            If a hexadecimal value is not being able to decode, raise the ValueError with the following text "non-hexadecimal number found"

 

  •  Define a test function called test1 which can do the following

 

            Pass the filename from file_sum1.txt,file_sum2.txt...file_sum5.txt using the pytest parameterize , and decode the hexadecimal values present in the file using decoder              function and compare them with the actual text present in the file.

    

  •  Define a test function called test2 which can do the following

 

            Pass the filename from file_sum6.txt,file_sum7.txt...file_sum10.txt using the pytest parameterize , and encode the actual text string present in the file using encoder              function and compare them with the hexadecimal values present in the file.
