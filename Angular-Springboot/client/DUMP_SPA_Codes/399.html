Given an array of integers, maximize points received as you delete elements using the following rules:

 

  1. Delete all elements of your chosen value.  Add the sum of those values to your points.
  2. Delete all elements equal to your chosen value plus or minus 1.  No points are scored with these deletions.

Once the entire array has been processed and there are no elements left, return the integer value of points calculated.

 

For example, the given array is elements = [5, 6, 6, 4, 11].  First, delete 11 for a score of 11.  Since there is no value 11-1 = 10 or 11 + 1 = 12, proceed with elements = [5, 6, 6, 4]. Choose to delete the two 6's for 12 more points. Delete any 6-1=5 and 6+1 = 7 elements: elements = [4].  Delete the 4 for 4 points. The total score is 11 + 12 + 4 = 27.

 

Function Description

Complete the maxPoints function in the editor below. The function must return a long integer denoting the maximum number of points you can earn by performing deletions.


maxPoints has the following parameter(s)

    elements: an array of integers

 

Constraints

  • 1 ≤ n ≤ 10 5
  • 1 ≤ elements[i] ≤ 105 , 0 ≤ i < n
Input Format for Custom Testing

Input from stdin will be processed as follows and passed to the function:

 

The first line contains an integer n, the number of integers in the array elements.

Each of the next n lines contains an integer describing elements[i] where 0 ≤ i < n.

 

Sample Case 0

Sample Input 0

3
    3
    4
    2

 

Sample Output 0

6

 

Explanation 0

Given elements = [3, 4, 2], we maximize our score by performing the following steps:

 

  1. Delete 4 to earn 4 points, which also deletes 4 − 1 = 3.  The array elements = [2].
  2. Delete 2 to earn 2 points and elements = [].

 

There are no more elements to delete, so the function returns the total number of points earned: 4 + 2 = 6.

Sample Case 1

Sample Input 1

6
    1
    2
    1
    3
    2
    3

 

Sample Output 1

8

 

Explanation 1

Given elements = [1, 2, 1, 3, 2, 3], we maximize our score by performing the following steps:

 

  1. Delete both the 3's to earn 3 + 3 = 6 points.  When this is done, the twos (3-1) must also be deleted for no points.  The array elements = [1, 1].
  2. Delete both the 1's to earn 1 + 1 = 2 points and elements = [].

 

There are no more elements to delete, so we can't perform any more steps. The function returns the total number of points earned, which is 3 + 3 + 1 + 1 = 8.
